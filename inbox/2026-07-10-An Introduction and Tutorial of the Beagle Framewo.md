# An Introduction and Tutorial of the Beagle Framework
> 原文: [https://arxiv.org/abs/2607.06731](https://arxiv.org/abs/2607.06731)

arXiv:2607.06731v1 Announce Type: new
Abstract: The Beagle framework is a GPU-based genetic programming framework that enables highly efficient genetic programming search using large population sizes by leveraging NVIDIA GPUs. This technical guide provides an introduction to the Beagle framework and provides detailed instructions for using the framework for symbolic regression problems.
